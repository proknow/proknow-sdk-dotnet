GetCredentialsStatusAsync()

Gets the ProKnow credentials (API key) status specified by the base URL and credentials (API key) asynchronously

Task<ProKnowCredentialsStatus> GetCredentialsStatusAsync()

Returns

Task<ProKnowCredentialsStatus>

The ProKnow credentials (API key) status

Examples

This example shows how to verify that the provided base URL and credentials are valid: 

using ProKnow;
using System.Threading.Tasks;

var pk = new ProKnowApi('https://example.proknow.com', './credentials.json');
var status = await pk.GetCredentialsStatusAsync();
if (!status.IsValid)
{
    throw new Exception($"Error validating credentials (API key): {status.ErrorMessage}.");
}</code></pre>

GetDomainStatusAsync()

Gets the ProKnow domain status asynchronously

Task<ProKnowDomainStatus> GetDomainStatusAsync()

Returns

Task<ProKnowDomainStatus>

The ProKnow domain status

Examples

This example shows how to verify that the ProKnow domain is up and reachable: 

using ProKnow;
using System.Threading.Tasks;

var pk = new ProKnowApi('https://example.proknow.com', './credentials.json');
var status = await pk.GetDomainStatusAsync();
if (!status.IsOk)
{
    throw new Exception($"Error connecting to ProKnow: {status.ErrorMessage}.");
}</code></pre>

Remarks

If the ProKnow domain is up and reachable, IsOK will be true regardless of the provided base URL and credentials.
