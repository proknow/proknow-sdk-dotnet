DownloadAsync(String)

Downloads this entity asynchronously as DICOM object(s) to the specified folder or file

Declaration
public abstract Task<string> DownloadAsync(string path)
Parameters
Type Name Description
System.String path

The full path to the destination folder or file
Returns
Type Description
Task<System.String>

The full path to the folder or file to which the file(s) were downloaded
Remarks
  • If the entity is an image set, the provided path must be an existing folder. A subfolder named {modality}.{series instance UID} will be created in this folder and the individual images will be saved to files named {modality}.{SOP instance UID}.dcm where {modality} is an abbreviation of the modality.
  • If the entity is not an image set and the provided path is an existing folder, the entity will be saved to a file named {modality}.{SOP instance UID}.dcm.
  • If the entity is not an image set and the provided path is not an existing folder, the entity will be saved to the provided path.
