Examples

This example shows how to find spatial registration objects for a patient that transform to or from a specific frame of reference: 

using ProKnow;
using System.Threading.Tasks;

 var pk = new ProKnowApi("https://example.proknow.com", "./credentials.json");
 var patientItem = await pk.Patients.CreateAsync("Clinical");
 var frameOfReferenceUid = "1.2.246.352.221.5093062159960566210763150553104377477";
 var sroSummaries = patientItem.FindSros(s => s.TargetFrameOfReferenceUid == frameOfReferenceUid ||
     s.SourceTargetFrameOfReferenceUid == frameOfReferenceUid);</code></pre>

This example shows how to find all spatial registration objects for a patient: 

using ProKnow;
using System.Threading.Tasks;

 var pk = new ProKnowApi("https://example.proknow.com", "./credentials.json");
 var patientItem = await pk.Patients.CreateAsync("Clinical");
 var sroSummaries = patientItem.FindSros(s => true);</code></pre>
