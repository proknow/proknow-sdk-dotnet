API

ProKnowApi is the root object for interfacing with ProKnow and includes the following properties for interacting with specific areas within ProKnow:

  • Audit - Interacts with audit logging
  • Collections - Interacts with collections
  • CustomMetrics - Interacts with custom metrics
  • Patients - Interacts with patients
  • Roles - Interacts with roles
  • ScorecardTemplates - Interacts with scorecard templates
  • Uploads - Interacts with uploads
  • Users - Interacts with users
  • Workspaces - Interacts with workspaces
