Examples

This example shows how to download a spatial registration object for a patient: 

using ProKnow;
using System.Threading.Tasks;

 var pk = new ProKnowApi("https://example.proknow.com", "credentials.json");
 var patientItem = await pk.Patients.CreateAsync("Clinical");
 var sroSummaries = patientItem.FindSros(s => true);
 var sroItem = await sroSummaries[0].GetAsync();
 await sroItem.DownloadAsync("DICOM");</code></pre>

Remarks

  • If the provided path is an existing folder, the spatial registration object will be saved to a file named REG.{SOP instance UID}.dcm.
  • If the provided path is not an existing folder, the spatial registration object will be saved to the provided path.
