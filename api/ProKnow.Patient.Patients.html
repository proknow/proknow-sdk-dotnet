CreateAsync(string, string, string, string, string)

Creates a patient asynchronously

public Task<PatientItem> CreateAsync(string workspace, string mrn, string name, string birthDate = null, string sex = null)

Parameters

workspace string

The ProKnow ID or name of the workspace in which to create the patient

mrn string

The patient medical record number (DICOM ID)

name string

The patient name

birthDate string

The patient birth date in the format "YYYY-MM-DD" or null

sex string

The patient sex, one of "M", "F", "O" or null

Returns

Task<PatientItem>

The created patient item

Examples

This example shows how to create a patient in the workspace called "Clinical": 

using ProKnow;
using System.Threading.Tasks;

 var pk = new ProKnowApi("https://example.proknow.com", "./credentials.json");
 var patient = await pk.Patients.CreateAsync("Clinical");</code></pre>

LookupAsync(string, IList<string>)

Looks up a collection of patients matching a given list of MRNs

public Task<IList<PatientSummary>> LookupAsync(string workspace, IList<string> mrns)

Parameters

workspace string

The ProKnow ID or name of the workspace in which to lookup patients

mrns IList<string>

The list of MRNs to look up

Returns

Task<IList<PatientSummary>>

A collection of patient summaries. If the MRN at a given index cannot be found, the result will contain will contain null at that index

Examples

Use this method to resolve a list of patient MRNs. Just provide the MRNs as a list for the second argument: 

using ProKnow;
using System.Collections.Generic;
using System.Threading.Tasks;

 var pk = new ProKnowApi("https://example.proknow.com", "./credentials.json");
 var patientSummaries = await pk.Patients.LookupAsync("Clinical", new List<string>() { "HNC-0522c0009", "HNC-0522c0013" });</code></pre>

Exceptions

ProKnowWorkspaceLookupException

QueryAsync(string, string)

Queries the ProKnow API for the collection of patient summaries

public Task<IList<PatientSummary>> QueryAsync(string workspace, string searchString = null)

Parameters

workspace string

The ProKnow ID or name of the workspace containing the patients

searchString string

If provided, returns only the patients whose MRN or name match the parameter

Returns

Task<IList<PatientSummary>>

A collection of patient summaries

Examples

This example queries the patients belonging to the Clinical workspace and prints the name of each patient: 

using ProKnow;
using System.Threading.Tasks;

 var pk = new ProKnowApi("https://example.proknow.com", "./credentials.json");
 var patientSummaries = await pk.Patients.QueryAsync("Clinical");
 foreach (var patientSummary in patientSummaries)
 {
     Console.WriteLine(patientSummary.Name);
 }</code></pre>

ProKnowWorkspaceLookupException
